[主題]
硬碟怎麼選？HDD vs SSD，M.2 vs 2.5 吋

[敘事脈絡]
「硬碟」這個詞其實包含三種完全不同的東西，大家說的不是同一件事，溝通起來很容易雞同鴨講：

HDD（機械硬碟）——舊式硬碟：
裡面有真實旋轉的磁盤，像一張微型黑膠唱片在高速轉動。
速度很慢（約 120 MB/s），容量大，適合存大量照片、影片備份。
2026 年台灣時價（原價屋實價）：
  1TB WD 藍標 NT$3,490 ／ 2TB Seagate 新梭魚 NT$3,950 起 ／ 4TB Seagate NT$5,450 起 / WD 藍標 NT$5,590
重要：2TB 跟 4TB 現在價差只剩 NT$1,500，多花一點就有兩倍容量。
→ 如果要買 HDD，直接買 4TB 才划算，2TB 已不值得。
致命缺點：拿來裝作業系統（Windows）會讓電腦開機超慢，不適合當主要硬碟。
結論：只買來當第二顆大容量備份槽；主要硬碟一定要用 SSD。
辦公 / 家用情境特別適合加裝 HDD：
  → 平常在電腦存放文件、相簿、影片的使用者，SSD 容量很快就不夠用
  → 加一顆 HDD 專門存這些東西，SSD 保持快速、HDD 負責大容量
  → 買就買 4TB（比 2TB 只多 NT$1,500，容量多一倍）

SSD M.2 NVMe（薄型，現代標準）：
外型像一根口香糖大小的細長條，直接插進主機板上的插槽。
速度極快（3,000–14,000 MB/s），是 HDD 的 30–100 倍。
這是 2024 年後所有新電腦的標準配備，組新機一定選這種。

2026 年台灣時價（TLC 顆粒，原價屋實價）：
  512GB TLC 約 NT$4,999–5,500 / 1TB TLC 約 NT$5,599–7,500 / 2TB TLC 約 NT$9,100–11,700
  ⚠ 以上為 TLC 顆粒價格，QLC 版本約便宜 NT$1,000–2,000，但不建議選（理由見下）

選購兩個重點：
① PCIe 4.0 vs 5.0
  PCIe 4.0：速度 3,000–7,000 MB/s，主流選擇，發熱量正常
  PCIe 5.0：速度 11,000–14,000 MB/s，速度更快但發熱量明顯更高
             → ⚠ Gen5 SSD 高速運作時溫度可達 70–80°C，散熱不足會自動降速甚至縮短壽命
             → 中階以上主機板通常內建 M.2 散熱片，直接蓋上即可，不需要另買 SSD 含散熱片版本
             → 主機板沒有內建散熱片才需要選 SSD 含散熱片款（約貴 NT$500–600）
             → 價格方面：同品牌同等級 Gen5 仍比 Gen4 貴；不同品牌之間比價會有落差，不代表 Gen5 整體更便宜
             → 高階 Gen5（Kingston FURY G5，1TB NT$12,800）貴一倍以上，一般使用者不需要
  → 結論：一般使用者選 Gen4 TLC 最省事；若選 Gen5，務必選含散熱片版本

② TLC vs QLC（顆粒類型）
  TLC（三層式）：耐用度高、持續寫入速度穩定，是目前的推薦標準
  QLC（四層式）：1TB 約 NT$4,330–4,750，但耐用度較低、大量寫入時速度會明顯下降
  → 直接選 TLC，不要為了省 NT$1,000 左右選 QLC

SSD 2.5 吋 SATA（厚型，舊世代）：
外型像一張信用卡但有厚度，要用排線連接主機板。
速度約 500 MB/s，比 NVMe 慢 6–10 倍，但比 HDD 快很多。
主要出現在舊型電腦或筆記型電腦的升級情境。
新組電腦通常不選這種，除非主機板沒有 M.2 插槽（極少見）。

2026 年組新機的儲存建議：
① 主要硬碟 → M.2 NVMe SSD（裝 Windows + 遊戲）
② 如果需要大量存照片 / 影片備份 → 加裝一顆 HDD
③ 2.5 吋 SATA SSD → 只在翻新舊電腦時用到

[主題]
為什麼 2026 年的電腦零件這麼貴？

[敘事脈絡]
在看配置和價格之前，先了解一件事：
為什麼 2026 年買電腦，感覺比幾年前貴那麼多？
背後有三個互相疊加的原因，每一個都在把價格往上推：

原因一：晶片製造成本爆炸
2026 年主流顯卡都採用台積電 3nm 或 4nm 先進製程。
一片 3nm 晶圓成本約 $19,500 美元（約 NT$600,000），
且良率（製造成功率）不到 100%，壞掉的晶片就是損失。
每張顯卡上的 GPU 晶片面積越大，成本越高——
RTX 5090 的 GB202 晶片面積達 750mm²，佔一整片晶圓很大比例。
結論：光是「把晶片做出來」的製造成本就已非常高，這是底線。

原因二：AI 伺服器把記憶體和產能「搶走了」
ChatGPT、Gemini、各家 AI 公司的伺服器需要大量 HBM（高頻寬記憶體）。
生產 HBM 和生產消費級 GDDR7 的廠商高度重疊（三星、海力士、美光）。
AI 訂單利潤更高 → 廠商優先給 AI → 消費級顯示記憶體和 DDR5 供貨減少 → 漲價。
台積電的先進製程產能也被 AI 晶片（H100、B100、A16 等）大量佔用，
消費級 GPU 的排程被往後推。
SSD 和 HDD 也受 AI 訓練的大量資料儲存需求帶動漲價。

原因三：NVIDIA 取消了 OPP 補貼計畫
過去 NVIDIA 有一個「開放合作夥伴計畫（OPP）」，
對零售商提供一定的補貼或保護，讓入門顯卡可以維持較低售價。
2025–2026 年 NVIDIA 悄悄取消這個計畫，
導致最便宜的新一代獨立顯卡從 RTX 5050 才剛起跳 NT$10,590，
原本應該存在的 NT$5,000–8,000 入門段已消失。

這三件事疊在一起的結果：
• 入門獨立顯卡 RTX 5050 最低 NT$10,590，沒有更便宜的選項
• 你如果不需要遊戲：買「帶內建顯示的 CPU」可以省下這筆費用
• 你如果需要遊戲：這個費用無法避免，只能選對 CP 值最高的一張

[主題]
套裝機 vs 自己 DIY，哪個划算？

[敘事脈絡]
這是買電腦最常見的第一個問題，但答案不是「哪個絕對好」，而是「看你是誰」。

套裝機（品牌整機）有兩種，缺點不同：

① 品牌大廠整機（ASUS / Acer / Lenovo / HP / Dell）：
  插電即用，整機保固，服務有保障。
  缺點：主機板是廠商自家定製 OEM 版，規格被鎖死，日後升級空間極小；
       電源可能是自家 OEM 特規版，品質尚可但無法直接對比零售品牌

② 組裝代工機（原價屋酷PC系列 / 欣亞整機套件等）：
  便宜有便宜的理由，不是詐騙，而是取捨——能不能接受，由你決定：
  SSD 可能是 QLC 顆粒（如 ADATA LEGEND 710，耐用度較低）、電源可能是較少見品牌（如 Apexgaming 銀牌）
  → 選前務必看規格欄：電源是哪個牌子、SSD 是否標明 TLC 顆粒

補充說明：PChome / MOMO 的主力商品是 ① 品牌大廠整機，鎖定完全沒有電腦概念的使用者，含正版 Windows 授權、插電即用。
