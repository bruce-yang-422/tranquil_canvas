# style
```text
整體風格是乾淨、冷峻、批判性的 1:1 square infographic，採用扁平化 editorial illustration 與資訊圖表混合風格。

視覺語言：
- 深色背景為主，使用炭黑 #1A1A1A、冷灰 #4A4A4A、冷白文字，以及少量警示紅 #C0392B 作為重點色。
- 圖像必須是抽象、象徵性、結構化，不使用真人照片、不使用真實人物肖像、不出現品牌 logo。
- 使用粗黑體感的繁體中文標題排版，標題清楚、留白充足、層級分明。
- 每張圖都要像社群輪播 infographic，可單獨理解，也能串成完整敘事。
- 構圖簡潔，避免過度複雜，畫面中保留明確文字區塊。
- 文字不要太多，重點字詞可放大處理，搭配簡單圖示、箭頭、對比欄位、時間軸或數據卡。

比例與格式：
- 1:1 正方形構圖
- 適合 IG / FB 輪播
- infographic poster style
- flat vector illustration
- high contrast
- clean typography layout
- Traditional Chinese text layout

禁止事項：
- 不要使用真實政治人物、企業家或名人的臉。
- 不要使用寫實血腥、墜落、自傷或災難畫面。
- 不要讓畫面像商業廣告。
- 不要出現錯字、亂碼、假 logo。
- 不要把批判性內容設計成正能量成功學風格。
```
