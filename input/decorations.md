# style
```
Soft, minimal illustration on a pure white or transparent background. Flat design with gentle organic shapes. Pastel colors, no hard outlines, no text, no people, no faces. Style: calm, therapeutic, watercolor-inspired with slight texture. Output should have clean edges suitable for compositing onto a colored background.
```
