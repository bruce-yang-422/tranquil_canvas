# image 05_taiwan_history
```
[主題]
臺灣郵筒演化史

[目的]
呈現臺灣從清代到現代的郵筒型態變遷脈絡

[版面]
垂直時間軸，5個節點由上而下

[元素]
節點1: 1906年 大清郵筒
  圓柱立柱、深綠配黃直條紋、圓形蘑菇狀帽蓋、正面紅底「大清郵政」標牌
  郵筒蓋可上掀，信差取出整個郵袋換裝，不會漏信

節點2: 1947年 趕班信筒
  圓柱造型、頂部一圈鮮黃帽、頸部淺藍色
  上午9點前投入保證當天送達；配合「航空運輸以郵件為先」政策
  郵局派機動車輛搶在班機起飛前沿街收信

節點3: 1952年 自助郵亭
  紅色涼亭造型，完全不像郵筒
  亭內陳列郵票、明信片、散幣，民眾自行購買投信
  無人看管——那個年代人人守法自律的時代象徵

節點4: 1956年 汽車郵箱
  掛在客運車上到處跑，不是固定立在路邊
  側邊印有「反共抗俄」、「發展鄉村郵遞」等字樣
  冷戰意識形態滲入日常郵政設施

節點5: 1973/1974年 方柱型雙筒
  1973年豔紅色方柱＝限時郵件；1974年墨綠色方柱＝普通郵件
  兩座分開立柱，顏色直接對應功能，沿用至今
  國際對比：英國、日本單一紅色；法國、德國單一黃色
  台灣功能分色雙筒並立，國際少見

每個節點附對應外型小圖示

[強調]
每個節點郵筒造型差異要一眼可辨
節點3自助郵亭造型最特殊，要畫出涼亭感而非信箱感
節點5要畫出紅綠兩座並排，標示各自功能
「國際少見功能分色」用小標籤點出

[避免]
不要讓節點文字全部等長，各節點輕重要有差異
不要把汽車郵箱畫成固定立柱
```
