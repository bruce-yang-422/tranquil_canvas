# 延伸企劃構想：工作電腦選購指南

> 接續「電腦購買懶人包」系列，針對有工作需求的使用者深入選購邏輯。
> 懶人包定位入門通用，本系列定位進階分眾——依職業用途拆開來說。

---

## 企劃定位

**目標讀者：**
平面設計師、動畫剪接師、工程 CAD 使用者、IT / 軟體工程師，
以及任何想一台電腦兼顧工作 + 本地 AI 的族群。

**與懶人包的分野：**
懶人包說「遊戲機的預算重心在顯卡」。
工作機的邏輯不同——依職業不同，重點可能在 CPU 核心數、VRAM 容量、RAM 大小、或 GPU 認證驅動。
沒有一個通用答案，要看你的工作是什麼。

---

## 開場議題：本地 AI — 跨所有職業的新變數

**這個議題放最前面，因為它影響所有領域的選購決策。**

2025–2026 年，本地 AI 工具（Ollama、LM Studio、Stable Diffusion、ComfyUI 等）
開始真正進入工作流程，不再只是玩具。
無論你是設計師、剪接師、工程師還是 IT 人員，都可能會碰到「想在自己電腦跑 AI」的需求。

### 核心概念：跑本地 LLM，VRAM 容量 > CUDA 核心數

跑本地語言模型時，「模型能不能塞進顯卡」比「推理速度快多快」更關鍵。
模型塞不進 VRAM，速度再快也沒用。

| VRAM | 可跑的 LLM 模型規模（參考） |
|------|---------------------------|
| 8GB  | 7B Q4（勉強）、3B Q8       |
| 12GB | 7B Q8、13B Q4             |
| 16GB | 13B Q8、34B Q4、70B 需量化 |
| 24GB | 34B Q8、70B Q4            |

**實務案例：**
RTX 5070（12GB）vs RTX 5060 Ti（16GB）
→ 純遊戲效能：5070 贏
→ 本地 AI 雙用途：5060 Ti 16GB 更務實，4GB VRAM 在大模型時是實質瓶頸

**其他 AI 用途的 VRAM 需求：**
- AI 圖像生成（Stable Diffusion / ComfyUI / Flux）：8GB 勉強跑 1024×1024，16GB 跑 Flux 更從容
- 語音辨識（Whisper 本地）：對 VRAM 要求低，4GB 就能跑
- Adobe AI 加速 / Blender 渲染：CUDA 核心數 + VRAM 兩者都需要

**AMD 顯卡提醒：**
ROCm 支援持續進步，但 2026 年社群工具兼容性仍遜於 NVIDIA CUDA。
純遊戲選 AMD 沒問題；有本地 AI 需求建議選 NVIDIA。

## 各職業用途選購邏輯

---

### 平面設計師
*Photoshop / Illustrator / Lightroom / Figma*

**重點零件排序：CPU > RAM > 顯卡**

| 零件 | 建議 | 理由 |
|------|------|------|
| CPU | 高頻多核（R7 / i7 以上） | Photoshop 濾鏡、Illustrator 複雜路徑吃 CPU 單核 |
| RAM | 32GB 起，習慣開多應用建議 64GB | 多個 PSD 同時開 + 瀏覽器 + 預覽軟體容易吃滿 |
| 顯卡 | 獨顯非必要，但有 GPU 加速 AI 功能（Neural Filters 等）效果更快 | Adobe AI 功能吃 VRAM |
| SSD | TLC 1TB 起，素材庫大建議加 HDD | 大尺寸 PSD 讀寫頻繁 |

**本地 AI 疊加需求：**
有在用 Stable Diffusion / Firefly 替代工具本地跑的設計師 → 建議顯卡上 16GB VRAM

---

### 動畫剪接師
*Premiere Pro / DaVinci Resolve / After Effects / Final Cut（Mac）*

**重點零件排序：GPU + CPU 並重 > RAM > SSD 速度**

| 零件 | 建議 | 理由 |
|------|------|------|
| CPU | R9 / i9 多核（12 核以上） | 轉檔、算圖、特效預覽吃核心數 |
| GPU | RTX 5070+ 建議 16GB VRAM | 4K 時間軸 + 特效層需要 VRAM；CUDA 加速解碼 / 轉碼 |
| RAM | 64GB | 4K 多軌 + After Effects 快取容易滿 |
| SSD | Gen4 TLC 2TB（主碟）+ HDD 大容量（素材儲存） | 影片素材動輒 TB 級 |

**特別注意：**
DaVinci Resolve 的 GPU 加速在 NVIDIA 上最完整（CUDA）；
AMD 顯卡可用但部分節點不支援，選前確認版本兼容性。

**本地 AI 疊加需求：**
AI 去背、超解析度、降噪等節點愈來愈多吃 VRAM，
剪接師有本地 AI 需求時 VRAM 16GB 是合理起點。

---

### 工程 CAD
*AutoCAD / SolidWorks / Revit / CATIA / Rhino*

**重點零件排序：CPU 單核 > 認證 GPU > RAM > SSD**

| 零件 | 建議 | 理由 |
|------|------|------|
| CPU | 高時脈（5GHz+），核心數其次 | AutoCAD / SolidWorks 主要單執行緒，單核速度直接影響操作順暢度 |
| GPU | NVIDIA RTX 系列（有 ISV 認證驅動） | SolidWorks / Revit 有 NVIDIA RTX 認證模式，穩定性更高；消費級 GTX/RTX 也能跑，但重度使用建議確認認證清單 |
| RAM | 64GB（大型 Assembly / BIM 模型） | 複雜組件、建築 BIM 全棟模型容易吃超過 32GB |
| SSD | TLC Gen4 1TB+（快速開啟大型檔案） | 大型 Assembly 開檔時 SSD 速度有感 |

**特別注意：**
工程 CAD 軟體授權費用通常遠超電腦硬體，
選購前先確認軟體廠商的 GPU 認證清單（部分軟體需要 Quadro / RTX 工作站認證版本）。

**本地 AI 疊加需求：**
目前 CAD 工作流程的本地 AI 整合較少，
但生成式 3D 建模（如 AI assisted SolidWorks）是未來趨勢，VRAM 16GB 做前瞻準備。

---

### IT / 軟體工程 / 資料科學

**重點零件排序：RAM > CPU 多核 > SSD > 顯卡**

| 零件 | 建議 | 理由 |
|------|------|------|
| CPU | 多核（R9 / i9，12–16 核） | 編譯、容器化（Docker）、跑多個虛擬機吃核心數 |
| RAM | 64GB 起，重度虛擬化建議 128GB | 多個 VM / 大型資料集 / 多工開發環境同時跑容易滿 |
| SSD | TLC 2TB（主碟），速度快讀取大型 dataset 有感 | 資料科學處理大型 CSV / Parquet 時 SSD 速度有感 |
| GPU | 有 ML 訓練需求：NVIDIA CUDA 必要；純開發可省 | PyTorch / TensorFlow 訓練靠 CUDA；推論可 CPU |

**本地 AI 疊加需求：**
IT / 資料科學族群最可能在本機跑 LLM（程式 Copilot、文件問答、私有資料 RAG）。
→ 這也是本地 AI 議題最直接的目標讀者之一。

## 雙用途取捨：工作 + 遊戲

工作機會打遊戲，遊戲機也能做工作。問題是預算怎麼分配。

**選卡方向：**

| 主要用途 | 建議顯卡 |
|----------|---------|
| 遊戲優先，偶爾跑 AI | RTX 5070 12GB（效能強，AI 受限） |
| AI / 工作優先，偶爾打遊戲 | RTX 5060 Ti 16GB（VRAM 充裕） |
| 兩者都重視，預算充足 | RTX 5070 Ti 16GB / RTX 5080 16GB |

**CPU 方向：**

| 主要用途 | 建議 CPU |
|----------|---------|
| 遊戲優先 | R7 7800X3D / 9800X3D（3D V-Cache） |
| 工作優先（剪接/渲染/編譯） | R9 / i9 多核 |
| 雙用途平衡 | R7 9800X3D（遊戲頂尖 + 多核不差） |

**RAM：**
- 純遊戲 → 32GB
- 工作機 → 64GB 起
- 雙用途 → 64GB（32GB × 2 雙通道）

## 2026 年市場現況

- DDR5 因 AI 需求漲價，工作機需要大容量 RAM 的預算壓力比遊戲機更大
- 高 VRAM 顯卡（16GB+）也因 AI 熱潮溢價，RTX 5060 Ti 16GB 從合理走向偏貴
- 目前非理想購機時期，但工作有剛需的使用者等不了
